# Smartphone Sales Analytics Dashboard

## Project Overview
This project provides an analysis of the smartphone retail market.
I developed an interactive Power BI dashboard to track KPIs, identify top-performing brands, and understand regional sales and time-based sales distribution across India.

The goal of this project was to transform raw sales data into actionable business insights regarding revenue growth, unit sales, and customer payment preferences.

## Key Features
* **Total Metrics Overview:** Real-time tracking of Total Sales Revenue, Total Units Sold, and Average Customer Ratings.
* **Brand Performance:** Comparison of market leaders (Apple, Samsung, Vivo, Xiaomi, OnePlus) by units sold and revenue generated.
* **Regional Analysis:** A geographical breakdown of sales across major Indian cities (Delhi, Mumbai, Bangalore, etc.).
* **Customer Insights:** Analysis of sales by age group and preferred payment methods (UPI, Credit Card, Cash, Debit Card).
* **Time-Series Trends:** Monthly and yearly sales trends to identify seasonal peaks.

## Key Insights from the Data
* **Brand Dominance:** The dashboard highlights which brands command the highest price per unit versus those that lead in volume.
* **Payment Trends:** A significant portion of transactions is driven by UPI and Credit Cards, reflecting modern digital payment adoption.
* **Demographic Targeting:** Sales are highest among specific age brackets, providing a clear target for marketing campaigns.

## Tech Stack
* **Tool:** Power BI Desktop
* **Data Source:** Excel/CSV
* **Data Transformation:** Power Query
* **Modeling:** DAX for calculated measures.

## 🖥️ How to View
1. Download the `.pbix` file from this repository.
2. Open it using **Power BI Desktop**.
3. If the data links are broken, point the data source to the `Mobile_Sales_Data.csv` file in the same folder.
